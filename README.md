✨ A modern, fast, and user-friendly e-commerce application designed to provide a seamless online shopping experience.

🚀 Features
 🛍️ Browse a wide range of products  
 🔍 Search and filter items بسهولة  
 📄 View detailed product descriptions  
 ➕ Add items to cart  
 🗑️ Remove or update cart items

 🔐 User authentication (Login / Signup)  
 📱 Fully responsive design (Mobile + Desktop)  
 ⚡ Fast performance and clean UI  

 🛠️ Tech Stack

  💻 Frontend: HTML, CSS, JavaScript  
  
▶️ Usage

 Open the application in your browser 🌐  
 Browse products by categories 🛍️  
 Use search to find items quickly 🔍  
 Add products to your cart ➕  
 Manage your cart (update/remove items) 🗑️  
 Proceed to checkout and place your order 💳
